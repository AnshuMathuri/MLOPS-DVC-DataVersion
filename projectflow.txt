1.create git repo and clone it in local
2.create a copy.py and add code to it .(it will save a csv file to a new "data" folder)
3.Do a git add -commit push before intializing dvc.
# pip install dvc
4. How we do "dvc init" (creates,dcvignore,dvc)
5. How do "mkdir s3" (creates a now s3 directory)
6. How we do "dvc remote add -d myremote s3"
7. Next "dvc add data/"
    Now it will ask to do:(git rm --cached 'data' and git commit -m "stop tracking data")
    Because initially we were tracking data/folder from git so now we remove it for DVC to handle.
8. Again we do "dvc add data/" (create data.dvc) then "git add .gitignore data.dvc"
9. Now - "dvc commit " and then "dvc push"
10. Do a git -commit-push to mark this stage as first version of data
11. Now make changes to code.py to append a new row in data , check changes via "dvc status"
12. Again - - "dvc commit" and then "dvc push"
13. Then git add-commit--push(we are saving v2 of data at this point)
14. check dvc/git status,everything should be upto date
15. Now repeat step 10-12 for v3 of data
